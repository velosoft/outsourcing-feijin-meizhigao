# 组件目录

### 项目主要结构
```
├── mp-html // 富文本解析
├── mpvue-citypicker // 地区选择器
├── AddImage // 图片上传
├── AgreementDialog // 协议弹框
├── DatePicker // 时间选择器
├── Dialog // 弹框
├── Empty // 数据为空的占位
├── FJImage // 图片标签的二次封装，用户开发阶段灵活的切换网络图片或者本地图片
├── ListContainer // 列表的无限加载
├── RightFilter // 筛选
├── SendCode // 发送验证码
├── NavBar // 头部导航栏
├── ShateTo // 分享卡片
├── Tabbar // tabbar栏
├── Tabs // tab栏
└── UpgradeDialog // 强制升级
```
